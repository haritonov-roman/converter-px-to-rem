Возможные форматы значений в px:
: 1px
: 10px
: 100px и т.д.
: 1PX
: 1Px
: 0.1px   - редкий случай применительно к px
: 10.10px 
            /\d{0,}\.{0,1}\d{1,}px/gi
            /\d*\.?\d+px/gi
: -1px - данный случай сводится к 1px
: transform(1px,1px) - данный случай сводится к 1px
: 1px 1px 1px 1px /- данный случай сводится к 1px



Алгоритм конвертации:
+ С помощью регулярного выражения забираем значения с пикселями;
+ Обрабатываем массив значений путем удаления px;
+ Производим конвертацию значений;
+ Добавляем к каждому значению rem;
+ Заменяем значения с px в исходном строковом массиве на значения с rem;
